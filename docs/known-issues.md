# 알려진 이슈 & 기술 부채

> 현재 구현 기준 남은 주의점·개선 여지. (해결한 이슈 기록은 issues.md)

---

## 운영 주의 (배포 전 체크)

1. **Redis 필수** — `KV_REDIS_URL` 없으면 메모리 폴백. 서버리스(Vercel)는 인스턴스마다
   메모리가 분리되고 재시작 시 사라져 → 초대/크레딧이 유실됨. 운영은 반드시 Upstash 연결.
2. **무료 체험 비용** — 서버 `GEMINI_API_KEY`로 돌아감(운영자 부담). 이미지 생성 5크레딧/장.
   AI Studio에서 **결제 상한** 꼭 걸 것.
3. **구글 콜백 URI** — 배포 도메인용 `/api/oauth/google/callback`을 구글 콘솔에 등록해야 로그인됨.
4. **AUTH_SECRET 분실 주의** — 바꾸면 기존 세션/초대 토큰이 전부 무효화됨.

---

## 설계상 한계 (의도된 트레이드오프)

- **멀티 구글계정 우회**: 체험은 이메일당 1회(`ever` 마커)지만, 새 구글 계정을 만들면
  새 체험 가능. 초대 링크 유효기간/삭제로 통제하는 구조.
- **초대 삭제/만료 차단 시점**: "즉시"는 다음 AI 호출/페이지 새로고침 시점. 이미 화면에
  떠 있는 결과물은 유지되고, 새 AI 호출만 막힘 (호출마다 초대 재확인).
- **BYOK 키 검증 없음**: 잘못된 키를 넣어도 입력 시점엔 통과, 실제 호출 시 Gemini 401로 드러남.
- **이미지 저장은 IndexedDB/localStorage**: 브라우저 데이터 삭제 시 작업물 소실 (드래프트 백업 기능 없음).

## 개선 여지 (선택)

- 초대별 **크레딧/기간 커스텀** (현재 전부 500/30일 고정 — `lib/trial.ts`)
- 관리자 페이지 **총 사용량/대략 비용** 대시보드
- BYOK 키 **입력 시 유효성 검사**
- 드래프트 **JSON export/import** (백업·이동)
- 첫 사용자 **온보딩 가이드**
- 모델 이미지 reference 없는 각도(예: 모자 뒷면) 추정 품질 — 가끔 디테일 차이

## 알려진 자잘한 것

- 개발 중 `ChunkLoadError`: 코드 수정 후 stale 청크 → 하드 리프레시로 해결 (코드 버그 아님).
- 큐텐 에디터는 `position:absolute`/base64 인라인 이미지를 제거 → 템플릿은 table 레이아웃,
  업로드는 큐텐 ZIP(슬라이스/Hybrid)으로 우회 (`lib/qoo10Export.ts`).
